# Pengucapan Awam & Spontan / Public & Impromptu Speaking

Dua komponen lisan PAC / Two PAC speaking components:
- **A) Pengucapan awam bersedia** — BM, 5 min, tajuk diberi ± 20 min awal.
- **B) Pengucapan spontan (impromptu)** — BM/English, 1 min fikir + 4 min cakap.

> Penilai menilai **isi + struktur + penyampaian + keyakinan**, bukan kesempurnaan. / Assessors score **substance + structure + delivery + confidence**, not perfection.

---

## A) Pengucapan Awam Bersedia / Prepared Public Speaking (BM · 5 min)

Tajuk diberi awal (± 20 min) — gunakan masa itu untuk **rangka, bukan hafal ayat penuh**.
Topic given ~20 min ahead — use it to **outline, not memorise word-for-word**.

### Rangka PEEL (setiap isi / per point)
- **Point** — nyatakan poin utama / state the main point
- **Explain** — huraikan / explain it
- **Example** — contoh / data / isu semasa / give an example or current fact
- **Link** — kaitkan balik ke tajuk / link back to the topic

### Struktur 5-minit / 5-minute structure
1. **Pembukaan (~30s)** — sapaan + tajuk + 3 perkara yang akan disentuh (peta jalan).
2. **Isi 1–3 (~3.5 min)** — setiap isi guna PEEL.
3. **Penutup (~45s)** — rumus 3 poin + kenyataan penutup yang kuat / seruan tindakan.

### Formula pembukaan / opening line
> *"Terima kasih tuan/puan pengerusi. Hari ini saya akan menyentuh [tajuk] melalui tiga perkara: pertama…, kedua…, dan ketiga…."*
> *"Thank you, chairperson. Today I will address [topic] through three points: first…, second…, and third…."*

Semua dwibahasa. / Combined: delivery craft (Tsar method) + 5 worked speeches, all bilingual.

## B) Pengucapan Spontan / Impromptu Speaking (BM/English · 1+4 min)

Cabaran sebenar = **struktur dalam 1 minit**. Jangan cari ayat sempurna — pilih **satu rangka** dan isi.
The real challenge = **structure in 1 minute**. Don't chase perfect words — pick **one framework** and fill it.

### 🔑 Dalam 60 saat persediaan / In your 60-second prep
1. **Ambil pendirian (10s)** — setuju / tak setuju / seimbang. *Pilih satu, jangan duduk atas pagar.*
2. **Pilih rangka (5s)** — PREP atau 4P (bawah).
3. **Cari 2 contoh (30s)** — 1 isu semasa Malaysia + 1 antarabangsa (lihat [`isu-semasa.md`](isu-semasa.md)).
4. **Tetapkan ayat penutup (15s)** — tahu bagaimana anda akan tamat.

### Rangka 1 — PREP (paling mudah / easiest)
| | |
|---|---|
| **P — Point** | Pendirian saya ialah… / My stand is… |
| **R — Reason** | Sebab utamanya… / The main reason is… |
| **E — Example** | Contohnya, [isu semasa]… / For example, [current issue]… |
| **P — Point** | Justeru, saya tegaskan… / Therefore, I reaffirm… |

### Rangka 2 — 4P / “Pandang segala sudut” (untuk isu dasar)
1. **Past / Latar** — apa isu ini & kenapa timbul.
2. **Present / Kini** — keadaan sekarang (fakta, kesan kepada rakyat).
3. **Problem-Solution / Cadangan** — apa boleh dibuat (peranan kerajaan/PTD).
4. **Prospect / Masa depan** — kesimpulan & harapan.

### Rangka 3 — Diplomatik (kuat untuk PTD) / Diplomatic frame
> Akui **dua sisi** → timbang → ambil pendirian seimbang. *Sesuai imej PTD yang matang.*
> Acknowledge **both sides** → weigh → land a balanced stand. *Fits the mature PTD image.*
> *"Di satu pihak…; di pihak lain…; namun secara seimbang, saya berpendapat…"*
> *"On one hand…; on the other…; on balance, however, I believe…"*

### Skrip "selamat" bila beku / Safety script if you blank
> *"Tajuk ini penting kerana ia menyentuh [rakyat/ekonomi/negara]. Saya melihatnya dari tiga sudut — [ekonomi], [sosial], dan [pentadbiran]…"*
> Tiga sudut serba boleh: **ekonomi, sosial, pentadbiran/tadbir urus.** / Universal three angles: **economic, social, governance.**

## 🎤 Teknik Penyampaian (kedua-dua) / Delivery (both tasks)
- **Kontak mata** merata; **bahasa badan** terbuka; senyum ringkas.
- Nada terkawal, **jeda untuk penekanan** — jangan laju. / Controlled pace, pause for emphasis.
- Buang pengisi: "ee…", "aa…", "macam". / Cut fillers.
- **Urus masa** — habis dalam had; latih dengan jam. / Time yourself.
- Sedia **dwibahasa**: latih tajuk sama dalam BM **dan** English.

## 🏋️ Rutin Latihan / Practice routine
- Tarik topik rawak → **1 min prep + 4 min cakap**, rakam video, tonton semula.
- Latih **5 tajuk/hari** bersilih BM ↔ English.
- Minta rakan beri **tajuk isu semasa** + tanya 1 soalan susulan.
- Bina **bank contoh** (10 isu semasa dengan 1 fakta + 1 pendirian) untuk dicabut bila-bila.
