# Ujian Psikometrik / Personaliti — Psychometric / Personality Test

Menilai personaliti, nilai, kecenderungan tingkah laku & kesesuaian dengan perkhidmatan awam. Selalunya **tiada jawapan "betul/salah"**, tetapi konsistensi dan ketulenan penting.
*Measures personality, values, behavioural tendencies & fit for the civil service. There is usually **no right/wrong answer**, but consistency and authenticity matter.*

## Jenis yang Mungkin / Possible Types
- **Inventori personaliti** (gaya kerja, kepimpinan, kestabilan emosi) / Personality inventory (work style, leadership, emotional stability)
- **Penghakiman situasi (SJT)** — pilih tindakan terbaik dalam senario kerja / Situational Judgement Test — pick the best action in a work scenario
- **Ujian aptitud** — penaakulan verbal, numerik, logik (jika ada) / Aptitude test — verbal, numerical, logical reasoning (if any)

## Prinsip Menjawab / Answering Principles
1. **Jujur tetapi konsisten** — soalan diulang dalam bentuk berbeza untuk kesan ketidakkonsistenan. / **Honest but consistent** — items repeat in different forms to detect inconsistency.
2. **Selari dengan nilai PTD** — integriti, kerja berpasukan, akauntabiliti, perkhidmatan, ketahanan. / **Aligned with PTD values** — integrity, teamwork, accountability, service, resilience.
3. **Elak jawapan ekstrem** melainkan benar-benar yakin; kebanyakan situasi menuntut keseimbangan. / **Avoid extreme answers** unless truly sure; most situations call for balance.
4. **Jangan "fake"** secara melampau — profil tidak realistik mudah dikesan. / **Don't over-fake** — an unrealistic profile is easily flagged.

## Untuk SJT — Tertib Keutamaan Lazim / For SJT — Usual Priority Order
1. Kepentingan awam & integriti dahulu / Public interest & integrity first
2. Patuh peraturan & rujuk pihak berkuasa bila ragu / Follow rules & escalate when unsure
3. Selesai masalah secara profesional & berpasukan / Solve problems professionally & as a team
4. Elak tindakan emosi, peribadi, atau melanggar etika / Avoid emotional, personal, or unethical actions

## Aptitud (jika ada) — Latihan / Aptitude (if any) — Practice
- Penaakulan numerik: peratus, nisbah, tafsir jadual/graf, kadar. / Numerical: percentages, ratios, reading tables/graphs, rates.
- Penaakulan verbal: benar/palsu/tidak dapat ditentukan daripada petikan. / Verbal: true/false/cannot-tell from a passage.
- Logik: urutan, corak, silogisme. / Logic: sequences, patterns, syllogisms.
- Latih kelajuan + ketepatan (had masa ketat). / Drill speed + accuracy (tight time limits).

> Berehat cukup malam sebelum — ujian psikometrik sensitif kepada keletihan & tergesa-gesa. / Rest well the night before — psychometric tests are sensitive to fatigue & rushing.
