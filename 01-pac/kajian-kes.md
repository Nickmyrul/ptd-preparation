# Kajian Kes & In-Tray Exercise / Case Study & In-Tray

Anda diberi satu situasi/masalah pentadbiran (atau set dokumen "in-tray") dan perlu menganalisis serta mencadangkan tindakan dalam masa terhad.
*You're given an administrative situation/problem (or an "in-tray" of documents) and must analyse it and recommend action within a time limit.*

## Rangka Analisis / Analysis Framework (gunakan secara konsisten / use consistently)
1. **Kenal pasti isu utama** — bezakan gejala vs punca. / **Identify the core issue** — symptom vs root cause.
2. **Pihak berkepentingan** — siapa terlibat & terkesan? / **Stakeholders** — who's involved & affected?
3. **Pilihan/opsyen** — senaraikan 2–3 pilihan tindakan. / **Options** — list 2–3 courses of action.
4. **Analisis pro & kontra** setiap pilihan (kos, masa, risiko, kesan). / **Weigh pros & cons** (cost, time, risk, impact).
5. **Cadangan** — pilih satu, justifikasikan. / **Recommendation** — choose one, justify it.
6. **Pelaksanaan** — langkah, jangka masa, KPI/pemantauan. / **Implementation** — steps, timeline, KPIs/monitoring.

## In-Tray: Keutamaan / Prioritisation
Bila diberi banyak tugasan/e-mel, susun ikut **Urgent–Important Matrix** / When given many tasks/emails, sort by the **Urgent–Important Matrix**:

| | Penting / Important | Tidak Penting / Not Important |
|---|---|---|
| **Segera / Urgent** | Buat dahulu / Do first | Wakil / Delegate |
| **Tidak Segera / Not Urgent** | Jadual / Schedule | Buang/tangguh / Drop/defer |

- Nyatakan **sebab** keutamaan anda. / State the **reason** for your priorities.
- Tunjuk anda boleh **mewakilkan** (delegate) tugas yang sesuai. / Show you can **delegate** suitable tasks.
- Ambil kira implikasi undang-undang/dasar & kepentingan awam. / Consider legal/policy implications & public interest.

## Nilai Perkhidmatan Awam / Public-Service Values (selitkan dalam jawapan / weave into answers)
- Kepentingan awam mengatasi kepentingan peribadi / Public interest over personal interest
- Ketelusan, akauntabiliti, integriti, anti-rasuah / Transparency, accountability, integrity, anti-corruption
- Patuh peraturan (Perintah Am, Arahan Perbendaharaan, tatakelakuan) / Follow rules (General Orders, Treasury Instructions, code of conduct)
- Tadbir urus baik & kecekapan kos / Good governance & cost-efficiency

## Latihan Contoh / Practice Example
> *Sebuah jabatan menerima aduan kelewatan kelulusan permit perniagaan, sementara kakitangan kekurangan dan belanjawan terhad. Anda Pegawai Tadbir baharu — cadangkan pelan tindakan 30/60/90 hari.*
> *A department faces complaints about slow business-permit approvals, amid understaffing and a tight budget. As a new Administrative Officer, propose a 30/60/90-day action plan.*

Latih: tulis jawapan berstruktur dalam **20 minit**, kemudian semak vs rangka di atas. / Practice: write a structured answer in **20 minutes**, then check against the framework above.
