# Rangka Esei Universal / Universal Essay Framework

Untuk **esei isu semasa** PAC (± **45 minit**, **satu bahasa** — BM **atau** English). Satu rangka untuk **mana-mana tajuk**: isi [tempat kosong] dengan tajuk anda.
*For the PAC current-issue essay (~45 min, one language). One framework for any topic — fill the [blanks].*

> Lihat juga / See also: [`esei.md`](esei.md) · bahan idea & fakta [`isu-semasa.md`](isu-semasa.md) · peranan PTD [`tentang-ptd.md`](tentang-ptd.md)

---

## ⏱️ Belanjawan Masa 45 Minit / 45-Minute Budget
| Fasa / Phase | Masa / Time |
|---|---|
| Faham soalan + rangka (mind-map) / Understand + outline | **~5 min** |
| Pengenalan / Introduction | ~5 min |
| Isi 1–3 / Body paragraphs ×3 | ~25 min (~8 min each) |
| Kesimpulan / Conclusion | ~5 min |
| Semak (tatabahasa, fakta) / Proofread | **~5 min** |

> Sasaran panjang: **5 perenggan, ~500–650 patah**. / Target: 5 paragraphs, ~500–650 words.

---

## 🧱 Struktur 5 Perenggan / 5-Paragraph Structure

### 1) Pengenalan / Introduction
**Tujuan / Goal:** tarik perhatian + definisi + **pernyataan tesis (pendirian)** + peta jalan. / hook + definition + **thesis (stance)** + roadmap.
- Cangkuk / Hook: fakta, soalan, atau senario. / a fact, question, or scenario.
- Definisi / Define the key term.
- **Tesis / Thesis** — pendirian jelas. / a clear stand.
- Peta jalan / Roadmap — 3 isi yang akan dibincang. / the 3 points to come.

> *"Isu [tajuk] semakin mendominasi wacana negara. [Definisi ringkas]. Esei ini berpendapat bahawa [pendirian], dengan menyentuh tiga aspek: [A], [B], dan [C]."*
> *"The issue of [topic] increasingly dominates national discourse. [Brief definition]. This essay argues that [stand], across three aspects: [A], [B], and [C]."*

### 2) Isi 1 · 2 · 3 / Body (gunakan formula PEEL setiap perenggan / use PEEL each)
- **P — Point / Ayat topik:** satu idea utama. / one main idea.
- **E — Explain / Huraian:** kenapa & bagaimana. / why & how.
- **E — Example/Evidence:** **fakta isu semasa + dasar negara** (RMK-13, Belanjawan, MADANI…). / a current fact + a national policy.
- **L — Link / Kaitan:** kembali ke tesis. / back to the thesis.

> *"Pertama, [point]. Hal ini kerana [huraian]. Sebagai contoh, [fakta/dasar — cth. BUDI95, NSS RM59.85b]. Justeru, [kaitan ke tesis]."*
> *"First, [point]. This is because [explanation]. For example, [fact/policy]. Therefore, [link to thesis]."*

**Tiga sudut serba boleh** (jika buntu) / Universal three angles (if stuck): **ekonomi · sosial · pentadbiran/tadbir urus**. / economic · social · governance.

### 3) Cadangan / Recommendations (boleh digabung dalam Isi 3 / can sit in Body 3)
- Tawar **2–3 penyelesaian konkrit** + peranan kerajaan. / 2–3 concrete solutions + the government's role.
- Akui **cabaran** & mitigasi (tunjuk kematangan). / acknowledge a challenge & mitigation.

### 4) Kesimpulan / Conclusion
**Tujuan / Goal:** rumus + tegas semula pendirian + pandangan ke hadapan. / recap + reaffirm + look ahead.
> *"Kesimpulannya, [rumus 3 isi]. Justeru, [tegas pendirian]. Demi [rakyat/negara], [seruan/harapan]."*
> *"In conclusion, [recap]. Therefore, [reaffirm stand]. For [people/nation], [call/hope]."*

---

## 🔗 Penanda Wacana / Connectors (silih-silih, jangan ulang / vary them)
- **Tambah / Add:** tambahan pula · selain itu · malah / furthermore · moreover · what's more
- **Lawan / Contrast:** namun · sebaliknya · walau bagaimanapun / however · yet · on the contrary
- **Sebab–akibat / Cause:** justeru · oleh itu · akibatnya / therefore · as a result · hence
- **Contoh / Example:** contohnya · sebagai bukti / for instance · as evidence
- **Rumus / Conclude:** kesimpulannya · secara ringkas · pada akhirnya / in conclusion · in short · ultimately

---

## ✍️ Kunci Markah Tinggi / What Scores High
- **Fakta & angka terkini** (Belanjawan 2026, RMK-13, statistik DOSM). / current facts & figures.
- **Pelbagai sudut** + **pendirian seimbang & matang** (akui sisi lain). / multiple angles + a balanced, mature stand.
- **Kaitkan dasar negara & peranan kerajaan/PTD.** / link to national policy & the government/PTD role.
- **Bahasa kemas** — ayat gramatis, kosa kata tepat, penanda wacana. / clean grammar, precise vocabulary, connectors.
- **Struktur jelas** — satu idea satu perenggan, ayat topik di hadapan. / clear structure, one idea per paragraph.

## 🧩 Templat Isi-Tempat-Kosong / Fill-in Template
- **Pengenalan:** [cangkuk] + [definisi] + **tesis: saya berpendapat [pendirian]** + "tiga aspek: [A], [B], [C]".
- **Isi 1:** [point A] → [huraian] → [fakta/dasar] → [kaitan].
- **Isi 2:** [point B] → [huraian] → [fakta/dasar] → [kaitan].
- **Isi 3:** [point C / cadangan] → [huraian] → [fakta/dasar] → [akui cabaran + mitigasi].
- **Kesimpulan:** [rumus 3 isi] + [tegas tesis] + [seruan/harapan].

## 🧪 Contoh Rangka Pantas / Quick Worked Outline
**Tajuk / Topic:** *"Subsidi tersasar: keadilan sosial atau beban pentadbiran?"*
- **Tesis:** subsidi tersasar ialah langkah berani ke arah **keadilan fiskal**, jika dilaksana dengan tepat.
- **Isi 1 (ekonomi):** subsidi menyeluruh membocor ke golongan mampu → BUDI95 kekal RON95 RM1.99 untuk ~84%, jimat RM2.5–4b/tahun.
- **Isi 2 (sosial):** penjimatan dialih ke kesihatan & pendidikan → lindungi B40/M40.
- **Isi 3 (pentadbiran + cabaran):** cabaran data kelayakan & komunikasi → perlu jaring keselamatan sosial.
- **Kesimpulan:** keadilan = memberi ikut keperluan; ujian kematangan negara.

*(English mirror: targeted subsidy is a bold step toward fiscal justice if well-executed — economic leakage, social redistribution, administrative challenge — justice means giving by need.)*

## 🔁 Rutin Latihan / Practice Routine
- **1 esei bermasa (45 min) setiap hari**, silih **BM ↔ English**. / one timed essay daily, alternating languages.
- Selepas tulis, semak vs **Kunci Markah Tinggi** di atas. / check against the scoring list.
- Simpan **bank fakta** (lihat [`isu-semasa.md`](isu-semasa.md)) untuk dicabut bila-bila. / keep a fact bank to draw from.
