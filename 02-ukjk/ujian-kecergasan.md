# Ujian Kecergasan Jasmani Kebangsaan (UKJK) / National Physical Fitness Test

> ⚠️ **Jenis ujian sebenar ditetapkan dalam LAMPIRAN C** e-mel SPA. Sahkan senarai & protokol rasmi di situ. Bahagian ini ialah persediaan umum berdasarkan bateri ujian kecergasan lazim. / **The actual tests are defined in ANNEX C** of the SPA email — verify the official list & protocol there. This is general preparation based on a typical fitness battery.

## Komponen Lazim Bateri Kecergasan / Typical Fitness Battery (jangkaan / expected)
| Ujian / Test | Mengukur / Measures | Persediaan / Prep |
|---|---|---|
| **Larian 2.4 km** atau **naik-turun bangku** / 2.4 km run or step test | Daya tahan kardiovaskular (VO₂max) / Cardiovascular endurance | Jogging/lari berkala 3–4×/minggu / Interval jogging 3–4×/week |
| **Tekan tubi (push-up)** / Push-ups | Kekuatan & daya tahan badan atas / Upper-body strength & endurance | Set harian, tambah ulangan / Daily sets, add reps |
| **Ringkuk tubi separa (curl-up)** / Sit-ups/curl-ups | Kekuatan teras / Core strength | Set harian / Daily sets |
| **Jangkauan melunjur (sit & reach)** / Sit & reach | Kelenturan / Flexibility | Regangan hamstring & belakang / Hamstring & back stretches |
| **BMI / antropometri** / BMI & anthropometry | Komposisi badan / Body composition | Jaga berat & pemakanan / Manage weight & diet |

*(Calon OKU: UAKK — Ujian Asas Keupayaan Kecergasan; rujuk Lampiran C. / Candidates with disabilities: UAKK basic fitness-ability test; see Annex C.)*

## Pelan Latihan 12 Hari / 12-Day Training Plan (22 Jun → 3 Julai)
Mulakan **sekarang**, naik beransur, **rehat 2 hari sebelum** ujian. / Start **now**, build gradually, **rest 2 days before** the test.

**Cardio (3–4×/minggu / 3–4×/week)**
- Minggu 1 / Week 1: jogging 20–25 min, atau interval 1 min laju / 2 min perlahan × 6. / jog 20–25 min, or 1 min fast / 2 min slow × 6.
- Minggu 2 / Week 2: cuba lari 2.4 km penuh, catat masa; ulang untuk perbaiki. / attempt a full 2.4 km, log your time; repeat to improve.

**Kekuatan / Strength (selang hari / alternate days)**
- Push-up: 3 set × maksimum (rehat 60s) — tambah 1–2 ulangan/set. / 3 sets × max (60s rest) — add 1–2 reps/set.
- Sit-up/curl-up: 3 set × 20–30.
- Plank: 3 × 30–60s.

**Kelenturan / Flexibility (setiap hari / daily)**
- Regangan dinamik sebelum, statik selepas. / Dynamic stretch before, static after.
- Sit & reach: regang hamstring 3 × 30s setiap kaki. / hamstring stretch 3 × 30s per leg.

## Hari Ujian / Test Day
- Sarapan ringan & berkhasiat 2 jam sebelum; **hidrat** dengan air. / Light, nourishing breakfast 2 hours before; **hydrate**.
- Pemanasan badan menyeluruh sebelum ujian. / Warm up thoroughly before the test.
- Pakaian: T-shirt **berkolar**, seluar track, kasut sukan. (Wanita: bukan legging/yoga pants.) / Attire: **collared** T-shirt, track pants, sports shoes. (Women: no leggings/yoga pants.)
- Bawa air, tuala, baju ganti. / Bring water, a towel, a change of clothes.

## Keselamatan / Safety
Jika ada masalah kesihatan serius (jantung, asma, darah tinggi, kecederaan lutut kronik, slipped disc, besi dalam badan) — pertimbang dengan teliti; ujian melibatkan kecergasan fizikal. Dapatkan nasihat perubatan jika ragu. / If you have a serious condition (heart, asthma, hypertension, chronic knee injury, slipped disc, metal implants), consider carefully — the test is physically demanding. Seek medical advice if in doubt.
