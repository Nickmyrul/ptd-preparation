# MyResume Checklist — DEADLINE 28 Jun 2026, 5:00 petang / 5:00 p.m.

🔗 **`myresume.spa.gov.my`** (Sistem MyResume / MyResume system)

> ⚠️ **PALING KRITIKAL & PALING HAMPIR.** Gagal lengkapkan = boleh jejaskan pertimbangan Suruhanjaya untuk ke peringkat temu duga. Maklumat palsu / tidak memenuhi syarat juga menjejaskan pertimbangan. / **THE MOST CRITICAL & NEAREST DEADLINE.** Failing to complete it can hurt your chances of reaching the interview. False or unqualifying information also counts against you.

## Dokumen Wajib Muat Naik / Required Uploads
- [ ] **i. MyKad / Kad Pengenalan** / Identity card
- [ ] **ii. Sijil SPM/SPMV/SVM** — *sijil sebenar, BUKAN slip keputusan* / the actual certificate, **NOT** the results slip
- [ ] **iii. Skrol** atau Surat Pengesahan Senat/Jawatankuasa (jika belum terima skrol) / Degree scroll or Senate/Committee confirmation letter (if scroll not yet received)
  - *Sertakan terjemahan BM/Inggeris jika bahasa lain digunakan. / Include a Malay/English translation if in another language.*
- [ ] **iv. Transkrip** Ijazah (atau mini transkrip jika belum terima skrol) / Degree transcript (or mini transcript if scroll not yet received)
  - *Sertakan terjemahan jika perlu. / Include a translation if needed.*
- [ ] **v. Kelulusan Badan Ikhtisas Perakaunan** (cth. ACCA) — *jika berkaitan sahaja* / Professional accounting body pass (e.g. ACCA) — *only if relevant*
- [ ] **vi. Pengiktirafan MQA** bagi kelayakan ijazah / MQA recognition of the degree
  - eSISRAF: `https://www2.mqa.gov.my/esisraf`
  - atau semak Daftar Kelayakan Malaysia (MQR) / or check the Malaysian Qualifications Register (MQR)

> **Hanya dokumen di atas** yang perlu dimuat naik. Jangan muat naik dokumen lain. / Upload **only the documents above** — nothing else.

## Tips
- Imbas (scan) semua dokumen **PDF jelas**, saiz fail munasabah, nama fail kemas. / Scan all documents as **clear PDFs**, reasonable file size, tidy filenames.
- Sediakan terjemahan awal jika transkrip/skrol bukan dalam BM/Inggeris. / Prepare translations early if not in Malay/English.
- Hantar **2–3 hari lebih awal** — elak masalah sistem pada saat akhir. / Submit **2–3 days early** — avoid last-minute system issues.
- Simpan tangkap layar pengesahan penghantaran. / Keep a screenshot of the submission confirmation.
